# Polestar Virtual Sound (가상 사운드 앱)

폴스타4용 6기통 엔진음 / 우주선 사운드 실시간 합성 앱 소스코드입니다.

## ⚠️ 먼저 꼭 읽어주세요 (중요한 한계)

1. **이 파일은 완성된 APK가 아니라 Android Studio 프로젝트 소스코드입니다.**
   컴파일 환경(Android SDK, Gradle, 네트워크)이 없는 곳에서는 실제 APK를 만들 수 없어서,
   본인 PC에 Android Studio를 설치해 직접 빌드하셔야 합니다.

2. **실제 차량의 RPM/속도(CAN 데이터)는 서드파티 앱이 읽을 수 없습니다.**
   Android Automotive OS는 차량 센서 데이터 접근을 구글이 승인한 시스템/파트너 앱에만 허용합니다.
   그래서 이 앱은 **GPS 속도**를 대신 사용해 RPM을 추정합니다 (정확한 실제 RPM과는 다릅니다).

3. **폴스타4 인포테인먼트에 사이드로딩하려면 개발자 모드/ADB 디버깅이 필요**하며,
   폴스타가 이를 공식 지원하지 않으므로 향후 소프트웨어 업데이트로 막힐 수 있습니다.

## 빌드 방법

1. Android Studio (최신 버전) 설치
2. `File > Open` 으로 이 폴더(`PolestarSoundApp`) 전체를 열기
3. Gradle Sync 완료 대기 (인터넷 연결 필요, 필요한 SDK/Gradle을 자동 다운로드)
4. `Build > Build Bundle(s) / APK(s) > Build APK(s)`
5. 생성된 `app-debug.apk`를 차량에 설치

## 차량에 설치하는 방법

- 폴스타4에서 설정 > 시스템 > 정보 화면을 여러 번 탭해 개발자 모드 활성화 (모델/소프트웨어 버전에 따라 지원 여부 다를 수 있음)
- USB 디버깅 활성화 후 PC와 연결, `adb install app-debug.apk` 실행
- 또는 파일 관리자 앱으로 APK를 차량 저장소에 복사 후 "알 수 없는 소스" 설치 허용

## 사용법

- 라디오 버튼으로 **6기통 엔진음**(기본값) / **우주선 사운드** / **끄기** 모드 선택
- 볼륨 슬라이더로 음량 조절
- "사운드 ON" 토글 시 위치/알림 권한을 요청하고, GPS 속도 추적을 포그라운드 서비스로 시작함
  → 속도에 따라 엔진 회전수(추정치)가 올라가며 사운드 피치/노이즈가 변함
- 정지 상태에서도 낮은 볼륨으로 아이들 사운드가 재생됨
- "사운드 ON"을 처음 누르면 **배터리 최적화 예외 허용** 팝업이 뜰 수 있는데, 반드시 **허용**을 눌러야
  화면이 꺼지거나 다른 앱(내비게이션 등)으로 전환해도 GPS/사운드가 끊기지 않음
  (특히 삼성 One UI는 백그라운드 앱을 매우 공격적으로 절전시키므로 이 설정이 중요함)
- 상태표시줄에 "가상 사운드 재생 중" 알림이 계속 떠 있는 게 정상 동작임 (포그라운드 서비스가 살아있다는 표시)

## 블루투스로 차량에 연결해서 테스트할 때 체크리스트

1. 휴대폰 설정 > 배터리 > 백그라운드 사용 제한에서 이 앱이 "제한 없음"으로 되어 있는지 확인
2. 위치 서비스가 "높은 정확도(GPS+Wi-Fi+네트워크)" 모드로 켜져 있는지 확인
3. 최초 실행 시 위치 권한 팝업에서 "앱 사용 중에만 허용" 이상으로 선택 (단순 "이번만 허용"은 재부팅/재시작 시 매번 다시 물어봄)
4. 소리는 휴대폰이 아니라 **차량 스피커(블루투스 미디어 오디오)로 라우팅**되는지 확인 — 통화용 프로필(HFP)만 연결되고 미디어 프로필(A2DP)이 안 붙어 있으면 소리가 차량으로 안 나갈 수 있음
5. 실제 폴스타4 인포테인먼트(Android Automotive OS)에 사이드로딩할 경우 이 앱은 시스템 앱이 아니므로
   여전히 GPS 기반 추정치를 사용하며, 차량이 자체적으로 앱을 백그라운드로 프리징하지 않는 한
   포그라운드 서비스 덕분에 계속 동작함

## 사운드 합성 원리

- **6기통 엔진음**: 4행정 6기통 엔진은 크랭크축 1회전당 3회 점화하므로 `점화주파수 = (RPM/60) × 3`.
  이 기본 주파수에 배음(2, 3, 4차)을 섞고, 저역통과 필터링한 화이트노이즈로 배기음 거친 질감을 더함.
- **우주선 사운드**: 72Hz/108Hz/144.5Hz로 살짝 디튠된 저음 오실레이터 3개를 겹치고,
  0.15Hz 느린 LFO로 피치와 볼륨을 흔들어 유기적인 SF 험(hum) 사운드를 만들고, 고역 쉬머 노이즈를 얹음.

## 커스터마이징 팁

- `SoundEngine.kt`의 `baseFreqs`, `harmAmp` 배열 값을 바꾸면 음색이 달라집니다.
- `maxSpeedForMapping`, `maxRpm` (MainActivity.kt)을 조절해 속도-RPM 매핑 감도를 바꿀 수 있습니다.
- V8, 로터리 등 다른 엔진음을 원하면 점화 배수 공식만 바꾸면 됩니다 (예: V8 4행정 = RPM/60 × 4).
